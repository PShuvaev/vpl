# vpl

vpl - это визуальный язык программирования с динамической типизацией и, одновременно, среда разработки. vpl позволяет составлять программы с помощью графических объектов, перенося их мышкой по принципу "drag-and-drop"(тащи-и-бросай).

Язык поддерживает управляющие структуры – последовательность, ветвление, цикл; стандартные операции (арифметические, присваивание и т. п.); вызов подпрограмм – как внешних, так и определенных пользователем.

Будучи реализованным на платформе Microsoft .NET Framework язык дает возможность простой интеграции со сторонними .net-библиотеками. Для создания vpl библиотеки необходимо создать .net-сборку, содержащую статические классы со статическими методами, возвращающими произвольные значения. Внимание! На данный момент не поддерживаются методы, возвращающие void. Для подключения библиотеку необходимо поместить в ту же директорию, что и исполняемый модуль и добавить в проект через интерфейс среды разработки.

Формат файла проекта совпадает с синтаксисом упрощенного C# языка и может быть откомпилирован стандартной утилитой csc.exe.
